# To Do Listen Programm

Ein einfaches To-Do-Listen Programm mit Python.

## Funktionen

- Aufgaben anzeigen
- Aufgaben hinzufügen
- Aufgaben löschen
- Aufgaben werden in einer Datei gespeichert

## Nutzung 

Starte das Programm mit:

Python To-Do-Listen-Programm.py

Danach im Menü auswählen:
- Aufgaben anzeigen
- Aufgaben hinzufügen
- Aufgaben löschen
- Beenden

## Beispiel 

To Do Liste:
1 = Anzeigen
2 = hinzufügen
3 = Löschen
4 = Beenden

Wähle eine Option: 2
Was möchtest du hinzufügen?
Hausaufgaben machen

Wähle eine Option: 1
Deine Aufgaben:
Hausaufgaben machen
